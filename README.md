# ⚽ Fuzzy Sports Recommendation System

A hybrid sports recommendation system based on fuzzy logic, clustering techniques, and similarity models, designed to support decision-making in football scouting and sports planning.

---

## 📌 Overview

This project implements an intelligent system that integrates:

- Fuzzy Logic (performance evaluation)
- K-Means Clustering (player segmentation)
- Fuzzy C-Means (multi-role membership modeling)
- K-Nearest Neighbors (player similarity)
- Spatial Analysis using KDE (heatmaps)

The system is deployed as an interactive web dashboard for real-time analysis and visualization.

---

## 🎯 Objective

To develop a data-driven tool that improves decision-making in football scouting by combining statistical performance, spatial data, and intelligent recommendation techniques.

---

## ⚙️ Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Visualization:** Chart.js
- **Data Processing:** R
- **Machine Learning:**
  - K-Means
  - Fuzzy C-Means
  - KNN (Euclidean + Cosine similarity)
- **Data Sources:**
  - StatsBomb Open Data
  - Transfermarkt

---

## 🧠 System Architecture

The system is composed of four main modules:

1. **Clustering Module**
   - Player segmentation using K-Means and Fuzzy C-Means

2. **Recommendation Engine**
   - Similar player detection using KNN

3. **Fuzzy Evaluation System**
   - Performance classification: Low / Medium / High

4. **Spatial Analysis**
   - Heatmaps using Kernel Density Estimation (KDE)

---

## 📊 Features

- Player similarity search
- Tactical profile analysis
- Fuzzy performance rating
- Interactive radar charts
- Heatmaps of player actions
- Optimal player search (based on constraints)
- Exportable scouting reports
